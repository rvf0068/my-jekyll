4. Code Snippets

Org-mode excels at code blocks. Here's a Python example: 

def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)

# Example usage
print(fibonacci(10))  # Outputs 55
